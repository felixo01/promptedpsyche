# Concepts UX Notes

## Current behavior

- Polish Concepts copy now uses `Słownik Prompted Psyche` instead of `Roboczy język pojęć`.
- The homepage teaser uses `Ważne pojęcia`.
- Related concepts are clickable on concept detail pages.
- Concept tags on `/concepts/` and `/pl/concepts/` filter the index with a `tag` query parameter.
- The previous decision to keep concept tags noninteractive has been superseded.
- Tags are now part of navigation through the glossary, not decoration.

## Scope

Article and note tags remain informational for now. They do not look or behave like primary navigation. Add article and note tag filtering only when there is a clear archive UX for those content types.
