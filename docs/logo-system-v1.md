# Prompted Psyche logo system v1

This is a practical website logo system v1 based on the approved raster reference board.

It is not a professionally exported final vector brand package. The mark has been redrawn as a simplified SVG approximation so the public website does not use a blurry screenshot or low-quality raster logo.

## Notes

- The icon keeps the Prompt Window direction: scan-frame corners, a cursor and a dotted right-facing human profile.
- The website header and footer use live text next to the SVG mark for readability and layout stability.
- The final designer-exported package can replace these SVGs later.
