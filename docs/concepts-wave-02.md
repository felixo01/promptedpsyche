# Concepts Wave 02

Wave 02 adds six public concepts to the Prompted Psyche concept library:

- Token
- Context window
- Cognitive load
- Mental model
- Metacognition
- Epistemic vigilance

The wave connects AI fundamentals with cognitive psychology, trust, verification and information work. The purpose is not to create a general AI glossary. The concepts stay tied to Prompted Psyche: AI, cognition, trust, communication and work.

## Editorial Scope

The public entries are compressed versions based on source-checked working briefs. Operational notes, internal readiness comments and planning material were not copied into the public repository.

Each entry includes:

- a short public definition
- why the concept matters
- the human-AI angle
- related concepts
- APA-style sources and context

## Held Back

Two candidate concepts were intentionally not published in this wave:

- AI agent
- tool use

Both require revision before a later public wave. The public definitions need to avoid vendor drift, automation hype and language that suggests human-like agency or intention.
